Question True-False

Answer true or false in each statement

Question 1

Transversal waves head to the same point

Feedback

False

Longitudinal waves head to the same point

Transversal waves go up and down

Question 2

The velocity of a wave is the time that a wave takes to complete a cycle

Feedback

False

The velocity corresponds to the propagation

The period corresponds to the time

Question 3

Pure destructive waves cancel each other

Feedback

True

Question 4

Waves cannot behaves as particles

Feedback

False

According to wave-particle duality, waves can behave as particles and vice versa

Question 5

Light behaves as particle and wave

Feedback

True
